BPM(Business Process Management)-negócios

1) Acesso do proprietario: Apenas o proprietario tera acesso ao site com uma conta de adiministrador

2)  Registro do cliente: Os clientes serão registrados no site a partir do usuario

3) Localização dos clientes no mapa: Os clientes serao localizados a partir de pontos de interesse presentes 
no mapa, para facilitar a busca o proprietario podera escolher os estados 

4) Atualização da localização: O proprietario ira atualizar as informações dos clientes 

5) Interface de monitoramento: O proprietario para poder acessar as informações dos clientes tera que 
clicar em seu ponto de iteresse, la ele poderar ver o nome da empresa, por qual organização e certificada,
qual estado fica, qual prefeitura fica, qual o seu iso (um ou mais), data da consultoria, o nome do consultor 
e o manual e o check list estão oks

BPM(Business Process Management)-sistema

1) Cadastro do Usuário: O proprietário terá uma conta de administrador já pronta,
   e a partir dela poderá criar outras contas de administrador, os chamados moderadores,
   com determinadas restrições sobre o acesso e controle dos dados, determinadas pelo administrador.

2) Deskboard do Proprietário: Após o login ser efetuado, o usuário será redirecionado ao mapa,
   que contém pontos de interesse, representando os clientes.

3) Acesso às Informações do cliente: O proprietário clica no ponto de interesse do cliente,
   localizado no mapa, usando o filtro de endereço ou procurando manualmente,
   que mostrará as informações do cliente selecionado.

4) Gestão de Clientes: O administrador podera adicionar novos clientes,
   editar ou excluir alguma informação de clientes já existentes,
   enquanto os moderadores somente poderão adicionar os clientes

5) Certificado ISO: O cliente pode checar se pode tirar uma ISO,
   se esta empresa estiver nos padrões para tirar tal ISO ou alguma outra,
   o cliente será informado pelo próprio sistema..
